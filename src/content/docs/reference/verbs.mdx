## Endings

The secondary endings are only used in the perfect indicative.

|  |  | primary | secondary | passive |
| - | - | - | - | - |
| **s** | **1** | -m / -o* | -i | -r |
|  | **2** | -s | -isti | -ris / -re |
|  | **3** | -t | -it | -tur |
| **p** | **1** | -mus |  -imus | -mur |
|  | **2** | -tis | -istis | -mini |
|  | **3** | -nt | -erunt / -ere | -ntur |

\**only used in the indicative mood in the present, future, and future perfect tenses of certain classes*

## Verb Formation

"<u>\*</u>" indicates a different first-person singular form

"<u>\*\*</u>" indicates a different second-person singular form

The "theme" category marking the "theme vowel" of a verb is not necessarily accurate, especially when it has subsumed/been subsumed by the affix.
